B.Tech. IT(Cloud Technology and Information Security)

About: The Bachelor of Technology in Computer Science and Engineering in Cloud Computing and Information Security is a four-year program broken up into eight semesters taught in conjunction with the knowledge partner iNurture. Candidates will learn about cloud computing, a model for delivering IT resources, including servers, storage, and applications, to end users over the internet. Topics covered in this course include virtualization, cloud computing, data centers, networks, and operating systems. Students will gain both theoretical and practical knowledge in information security thanks to this course. The course covers topics such as cryptography, ethical hacking, computer forensics, virtualization, and cloud security with an eye toward developing models, tools, and approaches for enforcing security policies.
